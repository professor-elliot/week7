Welcome to site1.local

This page is served directly by Nginx as static content.

Week 7 focus: virtual hosts and multi-site hosting.

When students reach this page, they should understand that Nginx matched a host identity and served a file from disk.

What this demonstrates

  • One web server can answer for more than one site
  • server_name helps identify which site should respond
  • Static content can be served without a backend application
